# About
This repository holds three mods for *Holder Of Place* and a template project file. Source code is in this repository. Downloads of the compiled code and image assets are below.
See [HolderOfPlace-ModdingBase](https://github.com/mhcdc9/HolderOfPlace-ModdingBase) for preparing the game for mods.

## Moths On Cards
[Download](https://github.com/mhcdc9/HolderOfPlace-Mods/releases/download/Initial-v0.1.0/Moths.zip) (updated 03/07/2026)

This mod adds ~~butterfly~~ moths to moth cards and Fate's Beloved. This has no effect on the gameplay.
## Unity Explorer
[Download](https://github.com/mhcdc9/HolderOfPlace-Mods/releases/download/Initial-v0.1.0/UnityExplorer.zip) (updated 03/07/2026)

This mod opens the Unity Explorer in-game. the Unity Explorer allows you to inspect every object in every scene to its minute detail. Very useful for mod development.
## Wildfrost Goats
[Download](https://github.com/mhcdc9/HolderOfPlace-Mods/releases/download/Initial-v0.1.0/Goats.zip) (updated 03/07/2026)

This mod adds a couple of cards based off the demon goats in Wildfrost. Art assets are not mine.

# Installing Mods
If you don't have the modding framework running yet, follow the instructions on the [other repository](https://github.com/mhcdc9/HolderOfPlace-ModdingBase) and then return here.

- In StreamingAssets folder that you set up in the other tutorial, locate the mods folder. 
- Place any of the mod folders downloaded above and place them into that mods folder. 
- Open the game and head to settings. You should see the mod icon of the right side. Click on the icon to turn it on/off. Click on the bars on the left/right to cycle through available mods.
